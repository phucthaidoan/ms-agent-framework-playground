# AI in C# using the Microsoft Agent Framework
Introduction course on the Microsoft Agent Framework in C#, teaching you:

- Zero to First Prompt
- Chat
- Tool Calling
- Structured Output
- Retrival Augmented Generation (RAG)

## Inspired by the Course

Inspired by [rwjdk/agent-framework-course](https://github.com/rwjdk/agent-framework-course), this repository adds more samples beyond the course.

## Build

Use the single solution file at the repo root for full builds and IDE work:

```powershell
dotnet build ms-agent-framework-playground.slnx
```

Run an individual lesson: `dotnet run --project samples/concepts/...` or `dotnet run --project samples/labs/...` (see `docs/sample-structure.md`).
